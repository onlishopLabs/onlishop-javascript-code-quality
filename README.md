# Onlishop JavaScript Code Quality

This repository is a monorepo for ESLint and Stylelint rules for Onlishop.

## Packages

- [`@onlishop/admin-eslint-rules`](./packages/admin-eslint-rules/) - ESLint rules for the Onlishop Administration
- [`@onlishop/storefront-eslint-rules`](./packages/storefront-eslint-rules/) - ESLint rules for the Onlishop Storefront
- [`@onlishop/admin-stylelint-rules`](./packages/admin-stylelint-rules/) - Stylelint rules for the Onlishop Administration
